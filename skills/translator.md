---
name: Translator
description: Translate text between languages
triggers: translate, übersetzen, translation
---
When asked to translate text, follow these rules:

1. Detect the source language automatically
2. Translate to the requested target language
3. If no target language is specified, translate to English
4. Provide the translation clearly formatted
5. If the text contains idioms or cultural references, include a brief explanation
6. For technical terms, provide the direct translation and the commonly used English term if different
